# SimpleBashUtils (C)

Моя реализация системных утилит Bash: **cat** и **grep**. 

### Что реализовано:
* **s21_cat**: поддерживает флаги -b, -e, -n, -s, -t, -v.
* **s21_grep**: поддерживает базовый поиск и флаги -e, -i, -v, -c, -l, -n.

### Технологии:
* Язык: **C** (стандарт C11).
* Сборка: **Makefile** (цели: all, s21_cat, s21_grep, clean, rebuild).
* Тестирование: написаны bash-скрипты для автоматической сверки вывода с оригинальными утилитами.

### Чему я научился:
1. Работа с системными вызовами и чтением файлов.
2. Парсинг аргументов командной строки (getopt).
3. Написание Makefile для автоматизации сборки.
